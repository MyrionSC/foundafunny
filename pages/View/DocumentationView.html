Documentation

Found a Funny is a synchronizong resource shower webpage. On the frontpage you can add new content, which is then shown on all open instances of the page. Content can be things in url format, such as an image, webm or youtube url. Content types, as well as other functionality, are explained below.


Content types

  • Resources: jpg, png, gif, webm
    • Putting in a file url will cause the files content to be displayed
  • API: Youtube, Twitch, imgur
    • Putting a url to one of the listed webservices will show content from the corresponding service
  • Commands
    • Coming Soon
  • Text
    • If nothing else fits, the content is parsed as text

Favorites

  • Favorites can be set by selecting the star next to the input bar. They can then be found in the favorites tab
  • Each item can easily be set as the current content

History

  • A list of the last contents can be seen in the History tab
  • Each item can be set as a favorite or set as the current content

Timers

  • A timer can be set in the Set Timers tab. At a chosen activation time the associated content will be selected
  • Timers come in a single timer variant, that only activates once, and a weekly timer variant, which will activate at the chosen days each week
  • It is possible to specify an end content as well, which will be selected a set number of seconds after the first content has activated
  • Multiple timer contents can be set. If this is the case, one of them will randomly be chosen when the timer activates
